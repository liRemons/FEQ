### React和Vue的区别

##### 相同点

- 虚拟DOM，数据驱动，组件化

##### 不同点：

- react使用fiber算法，vue采用diff算法
- react  jsx语法，vue 推荐html模板
- react 是单向数据流，vue双向数据绑定
- react  MVC，vue MVVM
